1. Install Xampp in your computer.
2. Extract the file in  computer/OS/xampp/htdocs.
3. Open xampp control panel and start Apache and Mysql server.
4. Open your default web browser and type localhost/, you will land on dashboard page, now click on phpMyAdmin at top right.
5. Click on new  from left panel and create a database named 'erp'.
6. Select the database and click on import button from the top bar.
7. Click on Choose file and select the SQL file named erp from extracted folder.
8. Click on GO.
9. Now type localhost/erp/ in your web browser. 
10. For online test portal admin username is 'admin@admin.com' and password is 'admin'.
